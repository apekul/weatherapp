
# Weatherapp

Current and forecast weather application 

## Features

- Current and forecast weather 
- City search
- Weather charts
- Map location display


## Demo

https://silly-gnome-94a7ea.netlify.app


## Tech Stack

React, TailwindCSS, Momentjs, Chartjs, ReactSimpleMaps, OpenWeatherAPI
